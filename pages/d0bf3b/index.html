dockerFile制作镜像脚本

# dockerFile是什么？

dockerFile是一组构建镜像的指令集，通过一条条指令描述镜像构建的过程

# dockerFile的指令

要清晰dockerFile的用法，就得先了解构建镜像指令的含义和文件结构、语法

# dockerFile 使用

通过docker build命令指定Dockerfile文件，构建镜像

# 基础镜像
FROM openjdk:8
# 维护者
MAINTAINER luke
# 镜像元数据
LABEL app="eureka" version="0.1" by="luke"
# 设置时间
RUN cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
# 设置工作目录
WORKDIR /app
# 复制jar包到镜像
COPY ./target/*.jar app.jar
# 期望的暴露端口
EXPOSE 8761
# 设置启动参数
ENV JAVA_OPTS="-server -Xms2g -Xmx2g -Xss512k"
# 根据镜像启动容器后执行的命令
CMD java $JAVA_OPTS -jar app.jar
