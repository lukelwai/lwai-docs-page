软考中级-软件设计师

# 1. 什么是软考？为什么想考这个？

其实在大学的时候我就有疑惑了，其他专业都需要考这个证考那个证的，难道我们程序员没有相关证件可以考？不应该吧？ 有的，关键还不少，其中我最关注的就是软考，这个国家人力资源和社会保障部、工业和信息化部联合组织实施的国家级考试。 那软考有什么好处呢？

  1. 国企事业单位，直接加薪，是的有工资加成，这个就很香
  2. 各个地区可以享受直接人才落户、落户加分政策
  3. 已经就业并且缴纳了失业保险满十二个月，可以凭软考证书领取人才补助，以及抵扣个税
  4. 部分证书(高级的信息系统项目管理师、中级的网络工程师、信息系统管理工程师)等有大量的挂靠需求，可以获得几千到上万每年的挂靠收入
  5. 最后软考还可以丰富提升我们个人的知识广度，对个人专业知识有提升作用

# 2. 软考-软件设计师介绍

软考分不同级别和不同方向，每个级别和不同方向的考试内容都存在一定的差异，而我选择的就是软件设计师方向中级考试。 该资格考试分为上午、下午；两个项目的考试，各设置两个半小时的考试时间， 其中，上午为客观题考试，有75个选择题，每题一分，满分75.下午为应用技术考试，四个必做题，两个选做题二选一，满分75，15分一题

# 1. 通过条件

上午题与下午题同时达到45及以上即为合格

# 2. 考试安排

每年两次，上半年在五月中旬考试，报名是在三月下旬，各个省份时间安排是不一样的，下半年是在十一月考试，报名是在七月或者八月

# 3. 涉及知识点

注意：其中上午的客观题中有五分的英文阅读理解是固定题型，c语言必须掌握，下午题必考，然后Java和c++要会其中一门(当然只要保证前面做的够好，不要这十五分也无所谓了。。)

  1. 掌握计算机内的数据表示、算术和逻辑运算方法;
  2. 掌握相关的应用数学及离散数学基础知识;
  3. 掌握计算机体系结构以及各主要部件的性能和基本工作原理;
  4. 掌握操作系统、程序设计语言的基础知识，了解编译程序的基本知识;
  5. 熟练掌握常用数据结构和常用算法;
  6. 熟悉数据库、网络和多媒体的基础知识;
  7. 掌握C程序设计语言，以及C++、Java中的一种程序设计语言;
  8. 熟悉软件工程、软件过程改进和软件开发项目管理的基础知识;
  9. 掌握软件设计的方法和技术;
  10. 了解信息化、常用信息技术标准、安全性，以及有关法律、法规的基础知识;
  11. 正确阅读和理解计算机领域的英文资料。

# 3. 备考

  1. 寻找相关资料文档，最好是官方的参考书什么的，不过不建议直接啃书
  2. 寻找学习视频教程，视频的形式可以更好的理解和学习
  3. 学习一定要做的事情就是输出，学习输出，笔记、博客都可以，还有就是刷题，无他唯手熟尔。
