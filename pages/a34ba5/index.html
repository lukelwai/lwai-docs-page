Linux常用命令

此处记录的命令以centOS 系统的Linux服务器为主

# 1. 常用命令

  1. cd，进入到文件夹
  2. cp，复制文件亦或者文件夹
  3. mv，移动文件亦或者文件夹
  4. mkdir，创建文件夹
  5. cat，以打印的方式查看文件内容
  6. vim，强大的Linux文件编辑器
  7. ps，查看系统进程
  8. top，进程面板
  9. sh，执行脚本命令
  10. tail，从文件内容的尾部，查看文件
  11. df -Th，查看磁盘容量大小
  12. grep， 过滤，可以和管道符一起用查询过来文件内容亦或者输出内容
  13. free -h，查看内存
  14. netstat -tunlp，查看端口占用情况
  15. pwd，查看当前路径
  16. find，查询

# 2. 经常用的重要命令

# 1. 系统信息

  1. cat /etc/redhat-release，查看系统版本
  2. cat /proc/cpuinfo，查看cpu信息

# 2. 防火墙

iptables
# 查看防火墙开放端口
firewall-cmd --list-all
# 新增防火墙开放端口：
firewall-cmd --zone=public --add-port=3306/tcp --permanent
# 开放端口后需要重新加载防火墙：
firewall-cmd --reload
# 删除端口 返回 success 代表成功
firewall-cmd --zone=public --remove-port=80/tcp --permanent
# 重新载入 返回 success 代表成功
firewall-cmd --reload

# 3. 解压

# 压缩文件
zip -r data.zip data
-r 递归压缩文件夹内的所有文件和目录
# 解压
unzip -o -d /home/sunny myfile.zip
# 把myfile.zip文件解压到 /home/sunny/
# -o:不提示的情况下覆盖文件;
# -d:-d /home/sunny指明将文件解压缩到/home/sunny目录下。

# 4. 用户权限操作

chmod u+x web.xml （---x------）		# 为文件拥有者（user）添加执行权限；
chmod g+x web.xml （------x---）		# 为文件拥有者所在组（group）添加执行权限；
chmod 111 web.xml  （---x--x--x）	# 为所有用户分类，添加可执行权限；
chmod 222 web.xml （--w--w--w-）		# 为所有用户分类，添加可写入权限；	
chmod 444 web.xml （-r--r--r--）		# 为所有用户分类，添加可读取权限；
